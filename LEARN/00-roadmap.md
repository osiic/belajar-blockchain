# 00. Roadmap & Instalasi

## Konsep
- Tooling utama: Node.js LTS, npm, Hardhat v2 (Runner), TypeScript opsional.
- Struktur Hardhat: `contracts/`, `scripts/`, `test/`, `hardhat.config.ts`, folder build (`artifacts/`, `cache/`).

## Langkah Praktik
1) Cek versi:
```bash
node -v
npm -v
npx hardhat --version
```
2) Install deps (jika belum):
```bash
npm install
```
3) Lihat konfigurasi: buka `hardhat.config.ts` dan catat versi Solidity.
4) Pastikan git bersih atau tahu perubahan aktif: `git status`.

## Selesai Ketika
- Perintah versi jalan tanpa error.
- Mengerti folder utama Hardhat dan fungsinya.

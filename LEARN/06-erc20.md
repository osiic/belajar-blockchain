# 06. ERC20 & Pembayaran

## Konsep
- Standar ERC20: `totalSupply`, `balanceOf`, `transfer`, `approve`, `transferFrom`.
- Minting vs fixed supply; allowance model.
- Uji transfer, allowance, kejenuhan (revert jika saldo kurang).

## Langkah Praktik
1) Install OpenZeppelin contracts:
```bash
npm install --save-dev @openzeppelin/contracts
```
2) Buat `contracts/LearnToken.sol`:
```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.28;
import "@openzeppelin/contracts/token/ERC20/ERC20.sol";

contract LearnToken is ERC20 {
    constructor() ERC20("Learn Token", "LRN") {
        _mint(msg.sender, 1_000_000 * 1e18);
    }
}
```
3) Tambah test `test/erc20.ts`: cek totalSupply, transfer, allowance.
4) Re-run compile + test.
5) Catat di notes: kapan pakai allowance vs direct transfer.

## Selesai Ketika
Token ERC20 berhasil diuji dan kamu paham alur approve/transferFrom.

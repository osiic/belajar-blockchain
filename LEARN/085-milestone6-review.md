# Milestone 6 Review

## Learning Objectives
- Mengevaluasi proyek P2E opsional.
- Menghimpun pelajaran game + anti-cheat.
- Menentukan apakah proyek layak lanjut atau dihentikan.

## Prerequisites
- Selesaikan modul sebelumnya.

## Theory
Game on-chain menantang validasi; cek biaya gas, UX, risiko kecurangan. Review cooldown, reward inflasi, dan langkah anti-cheat berikutnya.

## Hands-on Practice
1) Uji submit score beberapa kali (pastikan cooldown bekerja).
2) Review reward mint: pastikan tidak inflasi tak terkendali; hitung proyeksi supply.
3) Catat perbaikan di `notes.md` (anti-cheat, signer, rate limit).

## Exercises / Mini Project
- Putuskan lanjut/tidak proyek P2E; dokumentasikan alasan dan langkah jika lanjut.

## Quiz
- Q1: Apakah cooldown efektif?
- Q2: Apakah reward terkontrol?
- Q3: Langkah anti-cheat selanjutnya?

## Skill Checklist
- [ ] Review P2E selesai.
- [ ] Keputusan lanjut/stop dibuat.
- [ ] Risiko game dicatat.

# 12. Ekstensi & Karir Lanjut

## Ide Lanjutan
- Upgradeable: coba OpenZeppelin Upgrades plugin; pahami risiko penyimpanan.
- Observability: index dengan The Graph atau Subsquid.
- Otomasi: cron off-chain (Gelato) untuk perpanjang langganan.
- CI/CD: GitHub Actions untuk lint + test + deploy dry-run.
- Audit ringkas: checklist manual (akses kontrol, underflow, reentrancy, precision token).

## Praktik Cepat
1) Buat pipeline CI minimal: action yang menjalankan `npm ci` dan `npx hardhat test` pada push.
2) Catat daftar cek audit di `LEARN/notes.md` untuk diri sendiri.

## Selesai Ketika
Kamu punya pandangan jelas langkah berikut (upgrade, indexing, otomasi, atau audit) dan tahu resource lanjutan.

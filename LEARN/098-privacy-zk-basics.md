# Privacy & ZK Basics

## Learning Objectives
- Menjelaskan dasar zero-knowledge proof.
- Memahami perbedaan zk-SNARK vs zk-STARK (ringkas).
- Mengetahui use case privasi.

## Prerequisites
- Selesaikan modul sebelumnya.

## Theory
ZK = pembuktian tanpa membuka data. zk-SNARK: proof kecil, perlu trusted setup; zk-STARK: tanpa trusted setup, proof lebih besar, berbasis hash.

## Hands-on Practice
1) Baca artikel pengantar zk-SNARK/STARK.
2) Catat use case: private tx, verifiable computation, identity, anonymous voting.
3) Buat analogi sederhana di `notes.md` (misal amplop tersegel untuk pembuktian).

## Exercises / Mini Project
- Pilih satu use case privasi untuk proyek masa depan (contoh: verifikasi umur tanpa ungkap tanggal lahir).

## Quiz
- Q1: Apa inti ZK proof?
- Q2: Bedanya SNARK vs STARK?
- Q3: Contoh use case privasi?

## Skill Checklist
- [ ] Paham konsep ZK.
- [ ] Tahu perbedaan SNARK/STARK.
- [ ] Punya ide use case.

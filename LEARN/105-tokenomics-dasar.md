# Tokenomics Dasar

## Learning Objectives
- Memahami supply, inflasi, emisi.
- Mendesain utility token sederhana.
- Menilai distribusi awal.

## Prerequisites
- Selesaikan modul sebelumnya.

## Theory
Tokenomics mencakup supply cap, emisi, vesting, distribusi, utility (governance/fee). Distribusi adil mengurangi dump; vesting lindungi komunitas.

## Hands-on Practice
1) Rancang tokenomics LRN publik:
   - Cap: mis. 2,000,000 LRN.
   - Alokasi: 50% komunitas, 20% tim (vesting 1y cliff/2y vest), 10% liquidity, 20% ekosistem/grants.
   - Emisi: jika ada, definisikan per blok/bulan.
2) Buat tabel vesting di `docs/TOKENOMICS.md`.
3) Simulasikan saldo tim per bulan (sheet sederhana).

## Exercises / Mini Project
- Buat grafik alokasi pie + jadwal vesting untuk lampiran README.

## Quiz
- Q1: Mengapa vesting penting?
- Q2: Risiko inflasi tinggi?
- Q3: Utility apa yang menaikkan permintaan token?

## Skill Checklist
- [ ] Draft tokenomics dibuat.
- [ ] Vesting dipertimbangkan.
- [ ] Utility didefinisikan.
